# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v1.3.x  | :white_check_mark: |
| v1.2.x  | :x:                |
| < 1.2   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability in BugSneak:

1.  **Do NOT open a public issue.**
2.  Please email us directly at **security@bugsneak.com**.
3.  Include a description of the vulnerability and steps to reproduce.

We will acknowledge your report within 48 hours and work with you to resolve the issue promptly. We appreciate your responsible disclosure and will credit you for your discovery in our release notes.
